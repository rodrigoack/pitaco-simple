Como fazemos isso?

Análise & Documentação

- Avaliar ideias/projeto existente - Entrevistar responsáveis pelo projeto, em busca de compreender a necessidade e os objetivos - Contato com software/hardware existentes que possam já estar sendo utilizados - Documentar os casos de uso do produto atual - Testes de usabilidade no produto atual, para ver se se adaptam à necessidade dos usuários de maneira eficiente - Reuniões com os responsáveis para validação dos dados levantados e definição dos próximos passos

Avaliação & Melhorias

- Avaliar o budget e recursos disponíveis - Propor soluções com base no que foi levantado - Levantar e analisar a utilização de novas tecnologias, caso necessário - Treinamento para a equipe nas tecnologias propostas, caso se aplique

Novo Projeto!

- Se necessário for, elaboração de um novo projeto, utilizando nossa expertise - Documentação dos novos requisitos - Idealização de um MVP - Reuniões para validação dos dados levantados e definição dos próximos passos

Acompanhamento

- Acompanhamento e validação do desenvolvimento das soluções propostas

Mercado & Negócios

- Avaliação do modelo de negócio e, se necessário, novas propostas de modelo - Auxílio e aconselhamento na captação de investimento

O Céu é o Limite!

- Acompanhamento contínuo/esporádico do projeto desenvolvido
